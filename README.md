# TUGAS-9 DAN 10-PWEB-A

Pada tugas ini, kita diminta untuk membuat form pendaftaran siswa baru dengan mengimplementasikan backend menggunakan PHP. 

## Fitur dalam Web :
- Mendaftar siswa baru
- melihat daftar pendaftaran
- mengedit data
- menghapus data

## Tampilan utama Web (index.php)

![image](https://github.com/notdoppi/TUGAS-9-PWEB-A/blob/32d3684784a254e1a804065ba871f9493c564259/tugas%209/main%20web.jpg)

Gambar diatas merupakan tampilan awal ketika membuka web pendaftaran siswa. pada web tersedia 2 menu yaitu daftar baru (untuk mendaftar siswa baru) dan pendaftar (melihat data para pendaftar). 

## Daftar baru (form-daftar.php)

pada tab ini, kita bisa mengisi data-data untuk mendaftar. setelah klik `daftar`, maka data-data tersebut akan masuk ke dalam database.

![image](https://github.com/notdoppi/TUGAS-9-PWEB-A/blob/32d3684784a254e1a804065ba871f9493c564259/tugas%209/daftar%20baru.jpg)

## Pendaftar (list-siswa.php)

pada tab ini, akan teroutput data-data yang telah melakukan pendaftaran pada tab `Daftar baru`. pada tab ini, kita juga bisa untuk mengedit data dari data yang telah masuk ke dalam database dan juga menghapus data tersebut.

![image](https://github.com/notdoppi/TUGAS-9-PWEB-A/blob/32d3684784a254e1a804065ba871f9493c564259/tugas%209/pendaftar.jpg)

## Edit (form-edit.php)

pada tab ini, kurang lebih sama dengan tab `daftar baru`. tetapi, pada tab ini kita bisa memodifikasi data atau memperbarui data tersebut.

![image](https://github.com/notdoppi/TUGAS-9-PWEB-A/blob/32d3684784a254e1a804065ba871f9493c564259/tugas%209/edit.jpg)
